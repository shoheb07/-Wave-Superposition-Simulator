# -Wave-Superposition-Simulator
The Wave Superposition Simulator is a physics visualization project that demonstrates the principle of superposition of waves. The simulator displays two waves traveling through the same medium and shows the resulting wave formed by adding their displacements.

# Wave Superposition Simulator

## Overview

Wave Superposition Simulator is a physics-based visualization project developed using Python, Pygame, and NumPy. The simulator demonstrates the Principle of Superposition by displaying two moving waves and their resultant wave in real time.

The project helps students and educators understand how waves interact through constructive and destructive interference.


## Features

- Real-time wave animation
- Two independent sinusoidal waves
- Resultant wave visualization
- Constructive interference demonstration
- Destructive interference demonstration
- Smooth graphical rendering
- Educational physics simulation


## Technologies Used

- Python
- Pygame
- NumPy

Mathematical Model

Wave Equation:

y = A sin(kx − ωt)

Resultant Wave:

y_total = y₁ + y₂

Where:

* A = Amplitude
* k = Wave Number
* ω = Angular Frequency
* t = Time
* x = Position

  How It Works

1. Generate two sinusoidal waves.
2. Continuously update wave positions over time.
3. Calculate resultant displacement using superposition.
4. Render all waves on the screen.
5. Visualize interference patterns dynamically.
